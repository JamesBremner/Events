Detailed Description

A base timer to invoke a function at some later date.

This is an abstract base class. To use it must be specialized and the function handle_timer_event must be over-ridden

e.g.

class cTimedFunction : public raven::event::cTimer
{
public:
    void handle_timer_event()
    {
        // do something useful
    }
};
cTimedFunction tf;
tf.WaitThenPost( 1000 );    // do something useful 1 second from now
